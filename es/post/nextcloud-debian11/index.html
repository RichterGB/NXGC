Instalación de Nextcloud en Debian 11

En este tutorial, realizaremos la instalación de Nextcloud bajo el siguiente entorno:

  • Red: Privada (local)
  • Hardware: Raspberry Pi 4 8GB
  • Disco Duro: Adata 2TB
  • Sistema Operativo: Debian 11

Es posible ejecutar Nextcloud en un ordenador de uso general de bajo costo como la Raspberry Pi 4. Esta mini-computadora puede ejecutar Nextcloud aunque con ciertas limitantes causadas por el Hardware, sin embargo esto no es impedimento para su funcionalidad.

Arquitectura

La arquitectura arriba propuesta, no incluye los detalles de energizado / alimentación que es indispensable, no obstante el diagrama es bastante ilustrativo para el objetivo que se pretende.

Al conectar este mini-computador via ethernet al router de nuestro proveedor de servicio de internet, será posible establecer comunicación directa entre todos los dispositivos conectados a la red (cableada o inalambrica) de esta forma Nextcloud o nuestro servicio de almacenamiento estará disponible para cualquier dispositivo conectado a la red.

Instalar Debian 11 en Raspberry Pi 4 8GB

Debian a diferencia de Raspbian, es una distribución abierta, con las mismas capacidades que Raspbian o Ubuntu (distribuciones más populares). En este apartado se indicarán los pasos para instalar Debian 11 para Raspberry Pi.

  1. Descargar Debian 11
    • El enlace a la descarga de Debian 11 para Raspberry Pi 4 se encuentra aquí
    • Para otros modelos de Raspberry Pi cómo 3, 3B, 3B+, visite Daily auto-built images
    • Debian 11 es un proyecto mantenido por “Gunnar Wolf”, desarrollador debian.
  2. Descargar e Instalar localmente algún software de grabación de imagenes
    • Raspberry Pi Imager (Windows y Linux)
    • Balena Etcher (Windows, Linux y macOS)
  3. Conectar el disco USB a un equipo Windows, Linux o macOS con algún software de grabación de imagenes
